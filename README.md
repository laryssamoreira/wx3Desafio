# Project: Desafio Wx3
Projeto feito para teste de habilidades necessárias para vaga na empresa Wx3.

A estruturação do projeto foi feita de tal forma que, para futuras melhorias, fosse possível selecionar o frete desejado e fosse possível adicionar novas formas de pagamento.  
Optou-se pelo cliente fazer o cadastro e preencher o endereço de entrega em requisições separadas da venda para que, futuramente, fosse possível selecioná-los ou colocá-los em telas diferentes visando a organização da interface.
